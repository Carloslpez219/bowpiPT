# BowpiPT

## Descripción

## Instalación

Pasos para instalar y configurar el proyecto.

1. Clona el repositorio: `https://github.com/Carloslpez219/bowpiPT.git`
2. Entra al directorio del proyecto: `cd bowpiPT`
3. Instala las dependencias: `npm install`
